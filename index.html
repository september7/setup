setup.sh

setup.sh is a simple bash script, which installs the most important linux programs

what does it support?

currently, only debian is supported. we are working on porting it to arch and other distros

is the script finished?

no. we update it everyday

how do i run it?

download the repository, extract setup.sh and run sudo chmod +x setup.sh now run ./setup.sh

have an idea or found a bug?

please add a new issue to our repo
